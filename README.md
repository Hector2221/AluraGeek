# Alura Geek

Este es el repositorio del proyecto Alura Geek, desarrollado como parte del curso de Desarrollo Web de Alura.

## Descripción del proyecto

Alura Geek es una aplicación web que permite a los usuarios comprar productos geek, como figuras de acción, cómics, videojuegos, entre otros.

La aplicación cuenta con las siguientes funcionalidades:

- Autenticación de usuarios.
- Visualización de detalles de productos, incluyendo imágenes y descripción.
- Agregado de productos .
- Editar productos
- Eliminar productos
## Tecnologías utilizadas

El proyecto fue desarrollado utilizando las siguientes tecnologías:

- HTML
- CSS
- JAVASCRIPT
- API REST

## Instalación y uso

Para instalar y utilizar el proyecto, sigue los siguientes pasos:

1. Clona el repositorio en tu máquina local:

   ```
   git clone https://github.com/Hector2221/AluraGeek.git
   ```

2. Accede al directorio del proyecto:

   ```
   cd AluraGeek
   ```
## API
